Using an umbrella interview​

Preparing the included interview​

  1. remove all mandatory blocks in the child interview
  2. add a "named block" to the interview order block in the child interview
  3. give each attachment a variable name if it does not already have one

Reference the included interview in the umbrella interview​

  1. add a reference to the named block in the umbrella interview
  2. add the new variable name to the download screen using attachment code or ALDocumentBundle
